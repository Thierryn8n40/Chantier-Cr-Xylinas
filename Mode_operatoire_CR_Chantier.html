🔑 Accès & connexion

Le compte rendu est une page web unique, accessible depuis n'importe quel navigateur (PC, tablette, smartphone). Toutes vos modifications sont sauvegardées automatiquement en ligne — aucune installation n'est nécessaire.

  • 1
    Ouvrez le lienTapez (ou cliquez) l'URL ci-dessus dans votre navigateur. La page se charge en quelques secondes.
  • 2
    Entrez votre code d'accèsChaque entreprise dispose d'un code personnel fourni par le maître d'ouvrage. Saisissez-le dans le champ "Code d'accès" et cliquez Entrer.
  • 3
    Vous êtes connectéLe tableau de bord s'affiche. Votre nom d'entreprise apparaît en haut à droite.
⚠️ Code perdu ?

Contactez directement le maître d'ouvrage pour récupérer votre code. Ne partagez pas votre code avec d'autres personnes.

🏗️ Ce que fait une entreprise — étape par étape

Ajouter une annotation sur une tâche

  1. Allez sur l'onglet Compte rendu
  2. Trouvez la tâche qui vous concerne
  3. Cliquez le bouton 💬 Annoter sur la tâche
  4. Saisissez votre commentaire (avancement, question, information...)
  5. Cliquez Enregistrer → votre annotation apparaît immédiatement sous la tâche
💡 Bon à savoir

Chaque entreprise ne voit que ses propres annotations en mode édition, mais peut lire les annotations des autres. Votre annotation est horodatée et porte le nom de votre entreprise.

Mettre à jour votre planning

  1. Allez sur l'onglet Planning
  2. Repérez la ligne de votre entreprise
  3. Cochez (ou décochez) les semaines où vous êtes présent sur le chantier
  4. La sauvegarde est automatique

Consulter un document (plan, photo)

  1. Allez sur l'onglet Documents
  2. Utilisez le filtre "Phase" si nécessaire
  3. Cliquez 👁 Ouvrir sur le document souhaité — il s'ouvre dans un nouvel onglet

💾 Sauvegarde & synchronisation

La sauvegarde est entièrement automatique. Dès que vous enregistrez une action, annotation ou modification, les données sont envoyées sur le serveur cloud et instantanément disponibles pour tous.

  • Pas besoin de cliquer "Sauvegarder la page"
  • Pas besoin de renvoyer le fichier à qui que ce soit
  • Si vous travaillez à plusieurs en même temps, cliquez le bouton 🔄 (haut de page) pour actualiser et voir les dernières modifications des autres
💡 Où retrouver le CR à chaque visite ?

Toujours la même URL — marquez-la en favori dans votre navigateur. Chaque connexion charge automatiquement les données les plus récentes.

📤 Exporter et partager le compte rendu

(Admin et Assistante uniquement)

Exporter en HTML partageable (pour envoi par mail)

  1. Cliquez le bouton 📤 Exporter CR dans l'entête
  2. Un fichier CR_N01_DATE.html se télécharge automatiquement
  3. Joignez ce fichier à votre email — les destinataires l'ouvrent dans leur navigateur, sans connexion ni mot de passe

Imprimer ou créer un PDF

  1. Depuis le navigateur, faites Ctrl+P (ou ⌘+P sur Mac)
  2. Choisissez "Enregistrer en PDF" comme imprimante
  3. L'impression est optimisée format A4 — les menus et boutons sont masqués automatiquement

✅ Bonnes pratiques

  • Enregistrez l'URL en favori dès la première connexion (Ctrl+D)
  • Ne partagez pas votre code avec d'autres entreprises
  • Pour toute question sur une tâche, utilisez l'annotation plutôt qu'un email — la réponse sera visible de tous
  • Si la page semble "vieille" (données obsolètes), cliquez le bouton 🔄 Actualiser en haut
  • En cas de problème technique, contactez le maître d'ouvrage
